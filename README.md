Tasks Completed:

1. Hangman Game
Description:
A simple text-based game where the user guesses letters to find the hidden word. The player has limited attempts to guess the correct word.

Sample Output:

Enter a letter: p
Word: app__
Enter a letter: l
Word: appl_
Enter a letter: e

2. Stock Portfolio Tracker
Description:
This program calculates the total investment value of stocks. The user enters the stock name and quantity, and the program multiplies it with predefined stock prices.

Sample Output:
How many stocks do you want to enter? 2
Enter stock name: AAPL
Enter quantity: 2
Enter stock name: TSLA
Enter quantity: 3
Total Investment Value: 1110


3. Basic Chatbot
Description:
A simple chatbot that responds to user inputs like hello, how are you, and bye.

Sample Output:
Simple Chatbot
Type 'bye' to exit

Bot: Hi!
You: how are you
Bot: I'm fine, thanks!
